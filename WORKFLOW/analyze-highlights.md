---
description: Antigravityによるマルチモーダル解析とハイライト選別
---

このワークフローは、機械的に抽出されたハイライト候補に対し、Antigravity（AIエージェント）が動画の内容（文脈）を理解し、人間にとって面白いシーンを選別・構造化します。

// turbo-all

1. 候補データの読み込み
機械的解析の結果（`state.json` など）から、チャットの盛り上がりや発話サージがあったタイムスタンプを特定します。

2. クリップ動画のマルチモーダル解析
特定された各クリップについて、Antigravityが以下の観点で内容を把握します。
- 配信者の感情、リアクションの程度
- ゲームや雑談の「文脈」（何が起きて叫んでいるのか）
- 視聴者が「面白い」と感じるポイントの言語化

3. 規定の構造化フォーマット（`src/edit-config.json`）の出力
`doc/workflow.md` で規定された以下のフォーマットに従って、最終的なハイライト情報を出力します。

```json
[
  {
    "start_time_sec": <Number: 開始秒数>,
    "end_time_sec": <Number: 終了秒数>,
    "title": "<String: シーンの内容を表す簡潔なタイトル>",
    "video_src": "<String: 元動画の相対パス>",
    "telop_texts": [
      {
        "time_sec": <Number: テロップ表示開始秒数>,
        "text": "<String: AIが文脈から生成したテロップ>"
      }
    ]
  }
]
```
